# Titanic Data Analysis

## Overview
Exploratory data analysis of the Titanic dataset using Python.
Investigates what factors affected passenger survival.

## Key Findings
- Only 38% of passengers survived
- Female survival rate: 74% vs male survival rate: 19%
- 1st class passengers had highest survival rate (62%)
- Fare and passenger class were strongest predictors of survival

## Tools Used
- Python
- Pandas — data loading and cleaning
- Matplotlib — data visualisation
- Seaborn — statistical charts
- NumPy — numerical operations

## Files
- `day3.ipynb` — data loading, cleaning and analysis
- `day4.ipynb` — data visualisation and heatmap
- `titanic.csv` — the dataset
